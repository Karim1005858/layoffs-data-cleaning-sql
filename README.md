Layoffs Data Cleaning Project (SQL)
📌 Overview

This project focuses on cleaning and preparing a layoffs dataset using SQL.

🧹 Data Cleaning Steps
Removed duplicates using ROW_NUMBER()
Standardized text fields (company, industry, country)
Converted date format from text to DATE
Handled NULL values using self join
Removed irrelevant rows
🛠 Tools Used
SQL (MySQL)
📂 Files
data_cleaning.sql → contains all queries
🎯 Goal

Prepare clean data for further analysis
